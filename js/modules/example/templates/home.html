Dialog Simples

Mostrar Diálogo

O objeto global app pode ser utilizado para exibir um diálogo que recebe um objeto com os dados para exibição e dois callbacks. 


app.commands.execute("app:dialog:confirm", {

    icon: 'info-sign',

    title: 'Título do diálogo',

    message: 'Mensagem Importante para o usuário!',

    confirmYes: function() {console.log('yes');},

    confirmNo: function() {console.log('no');},

});
